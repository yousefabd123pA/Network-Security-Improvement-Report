# Jackson Corporation Security Assessment Report

## Project Overview

This project evaluates Jackson Corporation’s network and application security. It identifies existing vulnerabilities and provides practical recommendations to improve overall security.

## Objectives

* Assess network infrastructure security
* Identify security weaknesses
* Recommend practical improvements
* Apply defense in depth and least privilege principles
* Address OWASP Top 10 vulnerabilities

## Covered Areas

* Firewall architecture and segmentation
* Web server security
* Network monitoring and logging
* Incident detection and response
* Secure remote access
* Secure software development lifecycle
* Web application security (OWASP)

## Outcome

The project provides clear and practical recommendations to improve security, reduce risks, and strengthen the organization’s overall cybersecurity posture.
